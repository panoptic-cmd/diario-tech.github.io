1 - git init - iniciar git
2 - mkdir - criar pasta
3 - mv - mover ficheiro
4 - git add . - adicionar ao git
5 - git commit -m "" - commit / commentar
6 - git remote add origin <link> - fazer link ao repositorio no github
7 - git push -u origin master - adicionar do git para github
8 - git checkout -b <branch_name> - criar e mudar para nova branch
9 - cd - acesso a diretorio
10 - touch - criar ficheiro
